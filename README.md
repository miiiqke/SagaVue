SagaVue helps you navigate the gap between an anime and its source material. Search any episode, chapter, or arc to see exactly which manga chapters it adapts, filter out filler, track how far the anime has gotten, and follow the right watch order without getting spoiled.

The series catalogue is a work in progress. More titles are being added regularly and existing data is kept up to date.

## Features

- **Smart search** — search by episode number, chapter number, arc name, or just free text across all series at once. Try `aot s2e5`, `jjk ch200`, or `berserk ep14`.
- **Episode and chapter map** — every episode mapped to its exact manga chapters, with canon, filler, and mixed labels.
- **Adaptation progress** — see how far into the manga the anime has gotten and what's still unadapted.
- **Watch order guide** — step-by-step guidance for series with films, alternative adaptations, or skipped arcs.
- **Live MAL data** — scores, rankings, cover art, and related media pulled in real time from [MyAnimeList](https://myanimelist.net) via [Jikan](https://jikan.moe).

## Series catalogue

Currently includes: Attack on Titan, Berserk, Demon Slayer, Fullmetal Alchemist: Brotherhood, Frieren, Hunter x Hunter, Jujutsu Kaisen, and Vinland Saga.

More are on the way. To request a series or flag a data issue, open an issue on [GitHub](https://github.com/miiiqke/SagaVue/issues).

## Want to contribute?

No coding experience needed to add a new series. All it takes is filling out a JSON file and opening a pull request. The full walkthrough is in [`CONTRIBUTING.md`](CONTRIBUTING.md).
