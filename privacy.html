Privacy Policy

SagaVue is a fan resource. Here is exactly what we do — and don't do — with your data.

No data collection

SagaVue does not collect, store, or process any personal data. There are no user accounts, no sign-up forms, no login systems, and no sessions.

No cookies

We do not set any cookies — not for analytics, not for preferences, not for anything. Your browser stores nothing on our behalf.

No tracking

There are no analytics scripts, no advertising trackers, no pixel tags, and no third-party monitoring tools embedded in this site.

Third-party data

When you open a series page, SagaVue fetches publicly available data from the Jikan API — an unofficial read-only wrapper around MyAnimeList. This request is made from your browser directly to the Jikan API. No information about you is included in that request beyond a standard HTTP call. We have no control over Jikan's own privacy practices; you can review them at jikan.moe.

Google Fonts is loaded to serve the Fira Code typeface. This is a standard web font request; see Google's privacy policy for details. If you prefer, you can self-host the font and remove the Google Fonts link from the HTML.

Hosting

This site is hosted on Cloudflare Pages. Cloudflare may process standard server logs (IP address, request path, timestamp) as part of delivering the site. See Cloudflare's privacy policy for details.

Open source

SagaVue is fully open source. You can inspect every line of code on GitHub to verify these claims.

Contact

Questions? Open an issue on GitHub.
